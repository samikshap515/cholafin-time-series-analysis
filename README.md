
# CHOLAFIN Stock Time Series Analysis

## Objective
Analyze CHOLAFIN stock closing prices using ARIMA time series forecasting.

## Steps
1. Data preprocessing
2. ADF stationarity test
3. ACF and PACF analysis
4. ARIMA model implementation
5. 30-day forecast

p-value: 0.056248451604759234
